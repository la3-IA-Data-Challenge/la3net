Secure degree

Application permettant de mettre en œuvre un procédé de diffusion électronique sécurisée d'attestation de réussite/diplômes.
Il est notamment possible avec cette application de créer une demande de création d'attestation de réussite, de la valider/rejeter ou bien de vérifier l'authenticité de cette dernière.

Formulaire de demande d'attestation de réussite

Pour faire une demande d'attestation de réussite via un formulaire. Cette demande sera ensuite affichée dans la page affichant les liste des demandes.
